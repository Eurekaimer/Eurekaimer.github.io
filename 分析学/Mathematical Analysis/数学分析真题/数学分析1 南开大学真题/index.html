    2022级数学分析第1次月考

    一、

    用数列极限的定义证明：\(\lim\limits_{ n \to \infty } \frac{5n^{2}+2022}{n^{2}-n+1}=5\)
    Proof
    \(\forall\epsilon>0,\exists N=max\left\{ 2017,\left[ \frac{6}{\epsilon} \right]+1 \right\},n>N\)有
    $$
    \left|\frac{5n^{2}+2022}{n^{2}-n+1}-5 \right|=\left| \frac{5n^{2}+2022-5n^{2}+5n-5}{n^{2}-n+1} \right|=\left| \frac{5n+2017}{n^{2}-n+1} \right|
    $$
    分析：考虑将\(n^{2}-n+1\)改为\(n^{2}\),将常数2017改为n
    原式：\(\left| \frac{5n+2017}{n^{2}-n+1} \right|<\left| \frac{6n}{n^{2}} \right|=\frac{6}{n}<\epsilon\)

    二、

    设函数\(f(x)\)在\((0,b)\)有定义，\(\lim\limits_{ x \to 0^+ }f(x)=\infty\).证明：\(\forall a\in(0,b)\),\(g(x)=f(x)\cos \frac{1}{x}\)在\((0,a)\)上无界.
    Proof
    为了证明无界，可以利用题设的条件在0附近取值，将cos变为一个可放缩的值
    取\(x=\frac{1}{2n\pi},n\to \infty,x\to{0}\)
    那么\(\forall a,\exists n,s.t.0< \frac{1}{2n\pi}<a\),\(g\left( \frac{1}{2n\pi} \right)=f\left( \frac{1}{2n\pi} \right)\) ,再用极限语言陈述一下
    （\(\forall M>0,\exists \delta,\left| x-0 \right|<\delta,f(x)>M\)）

    三、

    设\(a\in \mathbb{R},x_{1}=a,x_{n+1}=x_{n}^{2}-x_{n}+1,n=1,2\dots\).证明：数列\(\{x_{n}\}\)收敛的充分必要条件是\(a\in[0,1]\)
    Proof
    这是统计与数据科学学院转专业考试的一道原题
    非常显然构造一个完全平方:\(x_{n+1}-x_{n}=(x_{n}-1)^{2}\ge 0\)
    首先得出单调性：这个数列是单调递增的
    \(\to\)
    \(A=A^{2}-A+1,A=1\),分情况讨论，若是\(a<0\)那么会有从第二项起的数列单项都大于1，矛盾
    若是\(a>1\),同理,再考虑0-1，存在上界1，可归纳
    \(\leftarrow\)
    同上，可以得出上界1，单调有界定理可以立得收敛

    四、

    $$
    x_{0}\in \mathbb{R},f(x)=\left{
    \begin{matrix}
    & \sin(\pi x), & x\in \mathbb{Q} \
    & 0, & x\not\in \mathbb{Q}
    \end{matrix}
    \right.
    $$
    证明：极限\(\lim\limits_{ x \to x_{0} }f(x)\)存在\(\iff\)\(x_{0}\in \mathbb{Z}\)
    Proof
    \(\leftarrow\)
    显然，分成两个讨论即可（有理和无理）
    \(\rightarrow\)
    已知存在，根据极限的唯一性，只有0可以作为极限（无理数的稠密性）
    而可以取到0的只有\(k\pi\)这些点，如果不是这些点，一定可以夹出来一个小的\(\delta\),使得极限不为0，换而言之，利用保号性。

    六、

    用现有知识计算极限\(\lim\limits_{ x \to 0 } \dfrac{xe^{x\cos}-\sin x}{\ln(1+x)\ln(\sin x+\cos x)}\)
    Solution
    $$
    \begin{align}
    \lim\limits_{ x \to 0 } \dfrac{xe^{x\cos}-\sin x}{\ln(1+x)\ln(\sin x+\cos x)} & =\lim\limits_{ x \to 0 } \dfrac{xe^{x\cos}-x+x-\sin x}{\ln(1+x)\ln(1+\sin x+\cos x-1)} \
    & =\lim\limits_{ x \to 0 } \dfrac{x(e^{x\cos}-1)+x-\sin x}{x(\sin x+\cos x-1)} \
    & =\lim\limits_{ x \to 0 } \dfrac{x(x\cos x)}{x^{2}}+\lim\limits_{ x \to 0 } \frac{x-\sin x}{x^{2}} \
    & =1+\lim\limits_{ x \to 0 } \frac{x-\sin x}{x^{2} } \
    & =1
    \end{align}
    $$
    注意，补充一些中间极限的证明：
    $$
    \lim\limits_{ x \to 0 } \frac{x}{\sin x+\cos x-1}=\lim\limits_{ x \to 0 } \frac{x}{\sin x-\frac{1}{2}x^{2}}=1
    $$
    $$
    \lim\limits_{ x \to 0 } \frac{x-\sin x}{x^{2}}=\lim\limits_{ x \to 0 } \frac{\tan x-\sin x}{x^{2}}=\lim\limits_{ x \to 0 } \frac{\sin x}{x^{2}\cos x} (1-\cos x)=\lim\limits_{ x \to 0 } \frac{1}{2}\tan x=0
    $$

    \(\{x_{n}\}\)是一个正数数列，\(\lim\limits_{ n \to \infty } \frac{x_{n}}{n}=0\),数列\(\left\{ \frac{x_{1}+x_{2}+\dots x_{n}}{n} \right\}\)有界，
    证明：\(\forall p>1,\lim\limits_{ n \to \infty } \frac{x_{1}^p+x_{2}^p+\dots+x_{n}^p}{n^p}=0\)
    Proof
    Think about: \(\lim\limits_{ n \to \infty }\left( \frac{x_{n}}{n} \right)^{p}=0\) ?
    \(\lim\limits_{ n \to \infty } \frac{x_{n}}{n}=0 \to \exists N,n>N, \frac{x_{n}}{n}<\epsilon,\left( \frac{x_{n}}{n} \right)^{p}<\epsilon^{p}<\epsilon(\epsilon<1)\)

    2023级数学分析第1次月考

    一、

    用函数极限的定义证明：\(\lim\limits_{ x \to 1 } \dfrac{x^{3}-1}{x^{2}-1}= \dfrac{3}{2}\)
    Proof
    $$
    \forall \epsilon,\exists\delta=min{\frac{1}{2},\epsilon},|x-1|<\delta,有
    $$
    $$
    \begin{align}
    \left| \frac{x^{3}-1}{x^{2}-1}-\frac{3}{2}\right| & =\left| \frac{(x-1)(x^{2}+x+1)}{(x-1)(x+1)}- \frac{3}{2}\right| \
    & =\left | \frac{(x^{2}+x+1)}{(x+1)}- \frac{3}{2} \right | \
    & =\left| \frac{2x^{2}-x-1}{2(x+1)} \right| \
    & =\left | \frac{2x+1}{2x+2} \right ||x-1| \
    & <\delta \
    & \le\epsilon
    \end{align}
    $$

    二、

    用现有知识计算极限\(\lim\limits_{ x \to 1 } \frac{(x^{2}-1)(\sin(\pi x))}{\ln ^{2}x}\)
    Solution
    利用等价无穷小（为了利用\(\ln x\)那么使用\(y=x-1\)）
    $$
    \begin{align}
    \lim\limits_{ x \to 1}\frac{(x^{2}-1)(\sin(\pi x))}{\ln ^{2}x} & =\lim\limits_{ y \to 0}\frac{(y+2)y(-\sin(\pi y))}{\ln ^{2}(y+1)} \
    & = \lim\limits_{ y \to 0 } \frac{(y+2)y(-\pi y)}{y^{2}} \
    & =-2\pi
    \end{align}
    $$

    三、

    叙述并证明海涅定理：
    Solution
    海涅定理：\(\lim\limits_{ x \to x_{0} }f(x)=A \iff \forall\{x_{n}\},x_{n} \to x_{0},x_{n}\neq x_{0},\)均有\(\lim\limits_{ n \to \infty }f(x_{n})=A\)
    Proof
    \(\rightarrow\)
    \(\forall\epsilon>0,\exists\delta>0,0<\left| x-x_{0}\right|<\delta,\left| f(x)-A \right|<\epsilon\)
    利用条件\(\exists N,n>N,0<\left| x_{n}-x_{0} \right|<\delta\),那么有\(\left| f(x_{n})-A \right|<\epsilon\)
    即\(\lim\limits_{ n \to \infty }f(x_{n})=A\)
    \(\leftarrow\)
    考虑使用反证法（由于\(x_{n}\)的任意性）
    假设\(\lim\limits_{ x \to x_{0} }f(x)\neq A\),即\(\exists\epsilon,\forall\delta,\exists x_{\delta},\left| x_{\delta}-x_{0} \right|<\delta,\left| f(x_{\delta})-A \right|\ge\epsilon\)
    考虑取\(\delta= \frac{1}{n}\),那么当\(\left| x_{\delta}-x_{0} \right|< \frac{1}{n}\),\(x_{n}\to x_{0}\),\(\lim\limits_{ n \to \infty }f(x_{n})=A\)
    与假设矛盾，那么\(\lim\limits_{ x \to x_{0} }f(x)\neq A\)

    四、

    设\(x_{n}=\sum\limits_{k=1}^{n} \frac{(-1)^{k}\sin k}{k^{2}+k-2023},n=1,2,\dots\),证明：数列\(\{x_{n}\}\)收敛
    Proof
    考虑Cauchy收敛原理，让\(n>2023\),将问题转换为\(\sum\limits_{n+1}^{n+p} \frac{1}{k^{2}}\)

    五、

    设𝑎 > 2, 令\(𝑥_1 = 𝑎^2\), \(𝑥_{𝑛+1} = (𝑥_𝑛 − 𝑎)^2\), \(𝑛 = 1, 2, · · ·\) . 证明：数列\({𝑥_𝑛}\)严格递增且\(\lim\limits_{ n \to \infty }x_{n}=+\infty\)
    Proof
    方法一：
    做数学归纳法，可以先证明\(x_{2}-x_{1}>0\),然后\(x_{n}-x_{n-1}>0\).
    $$\begin{align}
    x_{n+1}-x_{n} & =(x_{n}-a)^{2}-(x_{n-1}-a)^{2} \
    & =(x_{n}-x_{n-1})(x_{n}+x_{n-1}-2a) \
    & >4(x_{n}-x_{n-1})\end{align}
    $$
    \(x_{n}>4^{n-2}(x_{2}-x_{1})+x_{n-1}>4^{n-2}(x_{2}-x_{1})\)

    方法二：
    假设有界，可以根据单调有界定理，得出极限存在，那么利用关系式\(A=(A-a)^{2}\),可以得出\(A= \frac{(2a+1)+\sqrt{ 4a+1 }}{2}<a^{2}\)
    \((2a^{2}-2a-1)^{2}>(2a-1)^{2}=4a(a-1)+1>4a+1\)

    六、

    已知正数数列，a>0,\(\lim\limits_{ n \to \infty }x_{n}=a\),证明：\(\lim\limits_{ n \to \infty } \frac{\sqrt[n]{ x_{1}}+\sqrt[n]{x_{2} }+\dots\sqrt[n]{ x_{n} }}{n}=1\)
    Proof
    由于\(\lim\limits_{ n \to \infty }x_{n}=a\),那么有\(\lim\limits_{ n \to \infty }\sqrt[n]{x_{n} }=1\)
    再分段处理，利用\(Stolz\)定理(实质)
    $$
    \lim\limits_{ n \to \infty } \frac{\sqrt[n]{ x_{1}}+\sqrt[n]{x_{2} }+\dots\sqrt[n]{ x_{n} }}{n}=\lim\limits_{ n \to \infty } \frac{\sqrt[n]{ x_{1}}+\dots\sqrt[n]{x_{N} }}{n}+ \lim\limits_{ n \to \infty } \frac{\sqrt[n]{ x_{N}}+\dots\sqrt[n]{x_{n} }}{n}
    $$

    七、

    \[ \begin{align} & a_{n}=\sum\limits_{i=0}^{n} \frac{x^{i}}{i!} \\ & b_{n}=\left( 1+ \frac{x}{n} \right)^n \\ \end{align} $$ 证明：（用现有知识）$\lim\limits_{ n \to \infty }(a_{n}-b_{n})=0$ `Proof` 用逐项比较的方法 $$ \begin{align} \left| a_{n}-b_{n} \right| & =\sum\limits_{k=2}^{n} \left( \frac{1}{k!}- \frac{n!}{(n-k)!k!} \frac{1}{n^{k}} \right)x^{k} \\ & =\sum\limits_{k=2}^{n} \frac{1-\left( 1- \frac{1}{n} \right)\dots\left( 1- \frac{k-1}{n} \right)}{k!}x^{k} \\ & <\sum\limits_{k=2}^{n} \frac{1-\left( 1-\frac{k(k-1)}{2n} \right)}{k!}x^{k} \\ & =\sum\limits_{k=2}^{n} \frac{1}{2(k-2)!n}x^{k} \\ & = \frac{1}{2n}\sum\limits_{k=2}^{n} \frac{x^{k}}{(k-2)!} \\ \end{align} $$ $$ \begin{align} \lim\limits_{ n \to \infty } \left| a_{n}-b_{n} \right| & < \lim\limits_{ n \to \infty } \frac{1}{2n}\sum\limits_{k=2}^{n} \frac{\left| x \right| ^{k}}{(k-2)!} \\ & =\lim\limits_{ n \to \infty } \frac{ \frac{\left| x \right| ^{n+1}}{(n-1)!} }{2n+2-2n} \\ & =\lim\limits_{ n \to \infty } \frac{\left| x \right| ^{n+1}}{2(n-1)!} \\ & =0 \end{align} \]
