    Chapter 5 : Continuous Random Variables

    Definitions

    In Chapter 4, we have studied discrete random variables which take finite or
    countably infinite values. However, there exist random variables whose set
    of possible values is uncountable, for example, the lifetime of a transistor, the
    arrival time of a train at a specified stop.
    In this chapter, we consider the continuous random variable.

    discrete random variable -- probability mass function
    continuous random variable -- probability density function

    [!tip] Title
    there are random variables that are neither dicrete nor continuious

    [!NOTE]
    Definition 5.1
    A real-valued random variable \(X\) is said to be (absolutely) continuous if there exists a real-valued function f such that for any subset \(B ⊂ \mathbb{R}\),
    $$
    \mathbb{P}(X\in B)=\int _{B}f(x) \, dx
    $$
    Then f is called the probability density function (pdf) of the random variable \(X\).

    In particular, we let \(B = [a, b]\), we obtain that
    $$
    \mathbb{P}(a\le X \le b)=\int _{a}^b f(x) \, dx
    $$
    Recall that on probability space \((S, A, \mathbb{P})\), the (cumulative) distribution function
    (cdf) of a real-valued random variable \(X\) is the function \(F\) given by \(F(x) =\) \(P(X ≤ x)\), for any \(x ∈ \mathbb{R}\).

    [!NOTE]
    Definition 5.2 (cdf of continuous random variables)
    For a continuous real-valued random variable with pdf f, the distribution function is given by
    $$
    F(X)=P(X\le x)=\int _{-\infty}^x f(y)\, dy
    $$

    \[ F(x)=P(X\le x)=\left\{ \begin{matrix} & \sum\limits_{x_i\le x}p(x_{i}) & if \ X\ is \ discrete \\ & \int_{(-\infty,X]}f(y)dy & if \ X\ is \ conti nu ous \\ & \int_{(-\infty,X]}d \mu(y) & if \ X\ is\ neither\ discrete \ nor \ conti nu ous \end{matrix} \right. \]

    [!important]
    Remark 5.3
    We’ve discussed before that for the distribution function of any random variable,
    + F(x) is non-decreasing: for any \(x ≤ y, F(x) ≤ F(y)\).
    + \(\lim\limits_{ x \to -\infty }F(x)=0\) and \(\lim\limits_{ x \to \infty }F(x)=1\)
    + \(F\) is \(càdlàg\), that is, \(F\) is right continuous and has left limits.

    [!NOTE]
    Proposition 5.4 (Properties of pdf and cdf)
    If X is a continuous rrv, then
    1. For any \(a ∈ \mathbb{R}, \mathbb{P}(X = a) = 0\).
    2. For any \(a, b ∈ \mathbb{R}\) with \(a < b\),
    $$
    P(a ≤ X ≤ b) = P(a ≤ X < b) = P(a < X ≤ b) = P(a < X < b).
    $$
    3. \(F\) is the antiderivative of \(f\), and \(f\) is the derivative of \(F\), that is,
    $$
    F(x)=\int {-\infty}^{x}f(y) \, dy a n d f(x)=F'(x), if f is continu ous at x
    $$
    4. For any \(a, b ∈ \mathbb{R}\) with \(a < b\),
    $$
    \mathbb{P}(a ≤ X ≤ b) = F(b) − F(a).
    $$
    5. \(f\) is nonnegative on \(\mathbb{R}\), that is, \(f(x) ≥ 0\) for any \(x ∈ \mathbb{R}\).
    6. \(f\) is integrable on \(\mathbb{R}\) and
    $$
    \int     f(x)dx=1}^{\infty
    $$

    Proof

    1. \[ \mathbb{P}(X=a)=\mathbb{P}(X\in [a,a ])=\int_{{a}}^af(x)dx=0 \]
    2. All of them are \(\int_{a}^\mathbf{x}dx\)
    3. \[ F'(x)=\left( \int_{a}^x f(y)dy\right)'=f(x) \]
    4. By fundamental theorems of calculus \(F\) is an antiderivative
      $$
      P(X\in [a,b])=\int_{a}^b f(x)dx=F(x)|_{a}^b=F(b)-F(a)
      $$
    5. \(F\) is non-discreasing
      $$
      F'(x)=f(x)\ge 0
      $$
    6. \[ \int_{-\infty}^\infty f(x)dx=\mathbb{P}(x\in(-\infty,+\infty))=\mathbb{P}(x\subset \mathbb{R})=1 \]

    [!NOTE]
    Theorem 5.5
    A probability density function, or a cumulative distribution function completely
    determines the distribution of a continuous real-valued random variable.
    Two continuous rrv’s X or Y are said to be identically distributed, if they have
    the same pdf or cdf.

    [!NOTE]
    Definition 5.6
    A real-valued function f is said to be a valid pdf if the following holds:
    + \(f\) is nonnegative on \(\mathbb{R}\), that is, \(f(x) ≥ 0\) for any \(x ∈ \mathbb{R}\).
    + \(f\) is integrable on \(\mathbb{R}\) and
    $$
    \int _{-\infty}^\infty f(x) \, dx=1
    $$
    If \(f\) is a valid pdf, then there exists some continuous rrv \(X\) that has \(f\) as its pdf.

    [!example]
    Example 5.2
    If \(X\) is a continuous rrv with distribution function \(F_X\) and density function \(f_X\) .
    Find the density function of
    1. \(Y = 2X\).
    2. \(Z = X^2\)

    Solution

    1. We consider the distribution funtion \(Y\) for any \(x\in \mathbb{R}\)
      $$
      \begin{align}
      F_{Y}(x) & =\mathbb{P}(Y\le x)=\mathbb{P}(2X\le x)=\mathbb{P}\left( X\le \frac{x}{2 } \right)=F_{X}\left( \frac{x}{2} \right)
      \end{align}
      $$
      Taking derivative \(F_{Y}(x)\) , \(f_{Y}(x)= \frac{1}{2}f_{X}(x)\)
    2. For \(x\in \mathbb{R}^+\)
      $$
      \begin{align}
      F_{Z}(x)=\mathbb{P}(Z\le x)=\mathbb{P}(X^2\le x)=\mathbb{P}(-\sqrt{ x }\le X \le \sqrt{ x })=F_{X}(\sqrt{ x })-F_{X}(-\sqrt{ x })
      \end{align}
      $$
      Taking derivative \(f_{Z}(x)= \frac{1}{2\sqrt{ x }}f_{X}(\sqrt{ x })+\frac{1}{2\sqrt{ x }}f_{X}(\sqrt{ -x })\)

    Monotone function of random variable

    [!NOTE]
    Theorem 5.7
    Let \(X\) be a continuous rrv with probability density function \(f_X\) . Suppose that
    \(g(x)\) is strictly monotonic (increasing or decreasing), differentiable (and thus
    continuous) function of \(x\). Then the random variable Y defined by \(Y = g(X)\)
    has a probability density function given by
    $$
    f_{Y}(y)=\left{
    \begin{matrix}
    f_{X}[g^{-1}(y)]| \frac{d}{dy}g^{-1}(y)|=f_{X}[g^{-1}(y)]| \frac{1}{g'(g^{-1}(y))}| &if y=g(x) for so me x; \
    0 &if y\neq g(x) for a l l x;
    \end{matrix}
    \right.
    $$
    where \(g^{−1}(y)\) is the inverse function of \(g\).

    Proof

    We consider \(g\) is increasing
    For any \(y\in \mathbb{R}\) consider the distribution function of \(Y\)
    $$
    F_{Y}(y)=\mathbb{P}(Y\le y)=\mathbb{P}(g(X)\le y)=\mathbb{P}(X\le g^{-1}(y))
    $$
    if there exsits \(X\) s.t. \(y=g(x)=\int_{-\infty}^{g^{-1}(y)}f_{X}(x)dx=F_{X}(g^{-1}(y))\)
    Taking derivative
    we should know \(y'=(g(g^{-1}(y)))'\) \(1=g'(g^{-1}(y))(g^{-1}(y))'\)
    So we know \((g^{-1}(y))'=\frac{1}{g'(g^{-1}(y))}\)
    $$
    f_{Y}(y)=(g^{-1}(y))'f_{X}(g^{-1}y)=\frac{f_{X}(g^{-1}(y))}{g'(g^{-1}(y))}
    $$

    Expectation and Variance

    We have defined the expected value of a discrete random variable \(X\) by
    $$
    \mathbb{E}[X]=\sum_{x\in X(S)}xp(x)
    $$
    Now we can define the expected value of a continuous random variable.

    [!NOTE]
    Definition 5.8 (Expected value)
    Let \(X\) be a continuous rrv with pdf f. If \(\int_{-\infty}^\infty xf(x)dx\) is absolutely convergent, that is, \(\int_{-\infty}^\infty|x|f(x)dx<\infty\), then the expectation (or expected value or mean) of X exists, it is denoted by \(\mathbb{E}[X]\), and is defined as follows:
    $$
    \mathbb{E}[X]=\int _{-\infty}^\infty xf(x) \, dx
    $$

    Similarly as for discrete rrv, we can define the expected value of a function of
    a random variable.

    [!NOTE]
    Definition 5.9 (Expected value of a function of a random variable)
    Let \(X\) be a continuous rrv with pdf \(f\). Let \(g : \mathbb{R} → \mathbb{R}\) be a piecewise continuous function. If the random variable \(g(X)\) is integrable. Then, the expectation of \(g(X)\) exists, is denoted by \(\mathbb{E}[g(X)]\) and is defined as follows:
    $$
    \mathbb{E}[g(X)] = \int_{-oo}^\infty g(x)f(x)dx.
    $$

    [!NOTE]
    Proposition 5.10 (Properties of expectation)
    Let \(X\) be a continous rrv with pdf \(f\),
    1. for all \(c ∈ \mathbb{R}, \mathbb{E}[c] = c.\)
    2. If \(g : \mathbb{R} → \mathbb{R}\) is a nonnegative piecewise continuous function and \(g(X)\) is
    integrable. Then we have
    $$
    \mathbb{E}[g(X)] ≥ 0.
    $$
    3. If \(g_1 : \mathbb{R} → \mathbb{R}\) and \(g_2 : \mathbb{R} → \mathbb{R}\) are piecewise continuous functions and \(g_1(X)\) and \(g_2(X)\) are integrable. For \(c_1, c_2 ∈ \mathbb{R}\),
    $$
    \mathbb{E}[c_1g_1(X) + c_{2}g_{2}(X)] = c_{1}\mathbb{E}[g_{1}(X)] + c_{2}\mathbb{E}[g_{2}(X)].
    $$
    4. If \(g_{1} : \mathbb{R} → \mathbb{R}\) and \(g_{2} : \mathbb{R} → \mathbb{R}\) are piecewise continuous functions and \(g_1(X)\) and \(g_2(X)\) are integrable such that \(g_{1} ≤ g_{2}\). Then we have
    $$
    \mathbb{E}[g_{1}(X)] ≤ \mathbb{E}[g_{2}(X)].
    $$

    Proof

    1. \(g(x)=c\)
      $$
      \mathbb{E}[c]=\mathbb{E}[g(x)]=\int_{-\infty}^\infty cf(x)dx=c
      $$
    2. \(\mathbb{E}[g(X)]=\int_{-\infty}^\infty g(x)f(x)dx\ge 0\)
    3. \(c_{1}g_{1}(x)+c_{2}g_{2}(x)\) are integrable
      $$
      \mathbb{E}[c_{1}g_{1}(x)+c_{2}g_{2}(x)]=\int_{-\infty}^\infty(c_{1}g_{1}(x)+c_{2}g_{2}(x))f(x)dx=c_{1}\mathbb{E}[g(x)]+c_{2}E[g_{2}(x)]
      $$
    4. \(g_{2}(x)-g_{1}(x)\ge 0\) integrable ,Combining 2 and 3 gets the result

    [!NOTE]
    Definition 5.11 (Variance-standard deviation)
    Let X be a real-valued random variable. When \(\mathbb{E}[X^2]\) exists, the variance of \(X\) is defined as follows: \(Var(X) = \mathbb{E}[(X − \mathbb{E}[X])^2]\).
    \(Var(X)\) is sometimes denoted by \(σ^2_{X}\). The positive square root of the variance is called the standard deviation of X, and is denoted by \(σ_X=Var(X)\).
    An alternative formula is \(Var(X) = E[X^2] − (E[X])^2\)

    [!NOTE]
    Remark 5.12
    For a continuous rrv X with pdf f(x),
    $$
    Var(X)=\int_{-\infty}^{\infty}x^2f(x)dx-\left( \int_{-\infty}^{\infty}xf(x)dx \right)^{2}
    $$
    For \(a, b ∈ \mathbb{R}\),
    $$
    \mathbb{E}[aX + b] = a\mathbb{E}[X] + b, Var(aX + b) = a^2Var(X).
    $$

    Common continuous random variables

    The Uniform random variable

    [!NOTE]
    Definition 5.13
    We say that \(X\) is a uniform random variable on the interval \((α, β)\), denoted
    by \(X ∼ U(α, β)\), if the probability density function is
    $$
    f(x)=\left{
    \begin{matrix}
    & \frac{1}{\beta-\alpha} & if x\in(\alpha,\beta)
    \ & 0 & otherwise
    \end{matrix}
    \right.
    $$

    Validty

    \(f(x)\) here is valid, since \(f(x) ≥ 0\) by defintion, and
    $$
    \int_{-\infty}^\infty f(x)dx=\int_{\alpha}^\beta f(x)dx=1
    $$

    [!NOTE]
    Proposition 5.14
    Let \(X\) be uniformly distributed over \((α, β)\). Then
    + the distribution function is
    $$
    F(X)=\left{
    \begin{matrix}
    & 0, &if x\le a \
    & \frac{x-\alpha}{\beta-\alpha}, &if \alpha<x<\beta \
    & 1, &if x\ge\beta
    \end{matrix}
    \right.
    $$
    + \(\mathbb{E}[X]=\frac{\alpha+\beta}{2}\)
    + \(Var(X)=\frac{(\beta-\alpha)^{2}}{12}\)

    Proof

    1. obvious
    2. \(\mathbb{E}[X]=\int_{{\alpha}}^\beta x \frac{1}{\beta-\alpha}dx= \frac{\alpha+\beta}{2}\)
    3. \(\mathbb{E}[X^2]=\int_{{\alpha}}^\beta x^{2} \frac{1}{\beta-\alpha}dx= \frac{\beta^{2}+\alpha\beta+\alpha^{2}}{3},Var[X]=\mathbb{E}[X^{2}]-(\mathbb{E}[X])^{2}=\frac{(\beta-\alpha)^{2}}{12}\)
