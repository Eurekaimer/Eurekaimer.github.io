    2025年度总结

    2025年度总结

    2025年一定是我目前人生中非常痛苦的一年，我尝试去做了科研，结果并不好，尝试去拿到高GPA也是非常失败，想要去锻炼一直失败，一直生病，状态也并不好，一直内耗，经历了如此多的失败也没有崩溃或许是一个进步，因为我麻了可能，，，和很多人进行了一个短期social可能，然后发现social到的人都挺烂的，没话说直接屏蔽/删除

    下面大概通过一个灵魂之问进行回忆

    - 2025，我觉得最有意义的三个收获是什么？

    1. Passion with Linux Explorations(Now NixOS)
    2. First Step to Science Research(祛魅/扫盲)
    3. Yuri novels/comic/tv/ova/gal/广播剧(百广真王朝了)

    - 2025，我觉得最糟心的三个时刻是什么？

    1. 尝试第二次变e然后发现依然遇到的是烂人
    2. 发现周围的老师全是水货什么也学不到
    3. 发现很有信心的科目被打了低分直接无语

    我的年度总结几乎都是一样的步骤——先给出对过去一年的复盘和总结，然后展望新的一年立下下一年基本不会做到的Flag，今年大抵也是一样的，只不过今年我想要从几个印象比较深刻的点出发进行总结

    Elegant Linux

    Know about Linux

    我关于Linux的认识大概起源于高中和一些同学的闲聊，彼时我坚定的想要走科研学术道路，对于计算机方面除了算法兴趣平平，因此并未过多的深入了解，直到大二开始对于Linux系统有了想要了解的欲望，因为当时对于LLM/ML/DL有很大的兴趣，这种兴趣驱使我对于如何使用显卡/服务器进行代码模拟/模型训练进行学习，而牵扯到这些就很难不提到Linux，我碰巧是那种喜欢造轮子和折腾原理的人，因此很自然的开始接触一些基本的Shell命令和一些简单的逻辑

    关于这里值得宕开一笔说道说道，实际上在一开始我只是喜欢去学习一些算法诸如去看CS50x/CS61ABC这类课程，并且大二建立了自己的博客网站，但是对于诸如WSL/Docker/虚拟机等工具都没有什么认识，只是到了后来我发现使用Linux可以很好的优化我目前这台机器的性能并且很大程度的简化生活(起码我得到了几乎所有的电脑控制权)，因此最终在2025年底基本实现了全部的工作都可以由NixOS(我目前使用的Linux发行版)完成

    Learn to Use Linux

    关于Linux的使用是在和分布式打交道的时候逐渐成熟的，严格说来，在今年的下半年我才是基本做到了对于命令行比较熟悉并且可以基本自己解决遇到的所有的问题而不是通过看视频之类的

    从分布式的实验进行中大致就可以看出身边同学们对于电脑动手能力是非常差的，很多人根本不会使用命令行也不会使用检索工具去自己学习和改Bug，基本可以说感觉和白痴差不多(小白是对于什么都不会的人但是知道哪里可以找到答案的人的称呼而不是对于什么都不会的巨婴)，直到这里我才觉得，中国的计算机教育基本是一塌糊涂的，之前遇到的老师实际还会结合美国热门的教材并且使用国外课程进行改进优化，但是大三遇到的老师实在是一言难尽

    对于喜欢折腾的人来说，Linux就是一个取之无尽的宝库，你可以沉溺于美化你的桌面，你当然也可以想方设法去研究内核，你几乎可以通过代码去从0开始创作你使用的工具

    关于Linux的安装和使用配置，几乎可以说是赶上了好时代，我可以很轻松的通过GUI去安装几乎任何一个知名的发行版，几乎没有任何BUG需要修理，如果出现什么输入法爆炸，硬件驱动爆炸几乎都是因为你使用的不是最新版的教程和文档而已，连腾讯会议在今年都可以实现Linux本地化了(解决了困扰许久的屏幕共享问题，之前需要依赖别的项目，而那个项目也在今年Archived了)

    可以说Linux是真正可以Shock一个人的~玩具~系统，我几乎可以下这样的断言(我个人不是很喜欢断言类的语句)：如果一个人对于Linux的原理和折腾Linux毫无兴趣，他基本上对于CS的好奇度是很低的，我无法想象他从事CS相关行业是怎么样的(实际上我身边很多学习CS的同学快毕业了估计都不知道GitHub是什么，什么是Linux，类似之前Nips上那句"Who is Adam?")

    关于系统的一些思考

    不知道其他人有没有这样的一种感觉，重装系统可以让你很好的发现什么是你生活的重心。因为重装意味着你需要格式化你的硬盘(哪怕只是一部分)，你的的确确亲手杀死了那个硬盘里的所有文件，并将它们从鲜活的信息变成了无意义的01列，我重装了大概三四次系统(装失败的不计入在内)，可以很好的对于我现有的工作流有一个把握，我就不会再对于一些我不经常使用的软件纠结是否要保留了

    下面可以给出一些我的工作流软件用于参考，也方便我后续的整理：

    • Obsidian
    • Vscode(LaTeX and Python)
    • Clash/V2RayN
    • qq/wechat/wemeet/feishu

    Thinking About More Linux

    主要应该是围绕下面的几个方面去做

    1. 装一次Arch Linux体验一下传统btw，可以使用现有的一块硬盘，最好还有CachyOS去看看游戏的丝滑体验 体验AUR
    2. 安装多个桌面进行切换 大概有Hyprland/Niri/KDE/GNOME
    3. 尝试自己去打Nix包
    4. 更加深入的去学习Nix语言，我觉得实际上这门语言没有必须要学习的理由，如果不进行深层的应用的话

    关于为什么要安装一次ArchLinux，理由是简单的，因为ArchLinux的Wiki和软件包是做的非常好的，并且其理念也是相当激进我也很好奇，虽然我个人不喜欢那种不稳定的感觉因此我现在使用的系统是NixOS(u1s1，这个在你不小心把电脑搞爆炸之后重装起来只需要付出sudo nixos-rebuild switch花费的时间和打开科学软件的时间或者配置国内源的时间)

    至于多个桌面也很好理解，因为我是外貌协会的，我卡颜，而3、4则是想要深度的参与这个社区所必须要做的

    目前正在使用的是3N桌面：NixOS+Niri+Nictolia

    More Coding

    今年的Coding量应该是很大的，学到了很多新奇的技术，也算是对于编程解决问题和开发有了一些基本的认知了，包括一些学术论文代码的工程细节也有了一些感悟，这里面实在是水很深啊，想了一下大概就是看了很多项目然后发现基本都离不开Rust,包括我之前使用(现在也在使用)的Material for MkDocs也转向了新的使用Rust构建的Zensical(我之所以没有快速迁移到那里主要是因为它还有一些基本功能没有开发完整，例如博客系统等等，等开发的差不多了应该就会全部迁移过去了)

    然后进行一些基本的思考，今年主要是接触了下面的这些：

    • 利用torch复现顶会上的论文
    • 开始使用uv
    • 学会了Kaggle上面的GPU调用和关于核心调用的一些细节(比如奇怪的死锁等)
    • 写了Javascrips/HTML/CSS/JAVA这些
    • 算法题的训练 主要是Codeforces/LeetCode这些

    参加了一个Kaggle竞赛，大致对于如何优化性能和卷模型有了一些基本的了解，只不过那个竞赛对于初学者来说我认为还是有些困难了，因为性能增长有些缓慢，模型和算法也很难有新的花样

    下面是幸运种子加上别人的一个Baseline设计在某个竞赛中短暂拿到的金牌，虽然很快就被刷掉了但是还是很开心的

    Nice Games

    橘子洲之行

    关于近代中国绕不开的一个人一定是教员，他凭借他深邃的思想和伟大的人格魅力影响着一代又一代包括我在内的年轻人，他说我们是早晨八九点钟的太阳，他说知识分子要学会在实践中学习，他还说一切反动派都是纸老虎，我一直对于橘子洲上他的雕像非常神往，于是在本次期末结束之后就欣然成行(非常激动)，对于我来说是一次非常珍贵的朝圣经历了

    下面大概就对于长沙之行做一个游记的概括，我原本想过要单独开出一篇进行，但是我后来想了一想这也是我这一年很重要的一个时刻，我不想简单的把这个经历放在旅行中一笔带过，于是就干脆将它放在年度总结中

    关于出路的迷茫

    成绩大概是出来了不少了，有一门十分的差，几乎可以说是我上学以来拿到过的最差的成绩，我甚至觉得在外面面试，别人看到我的成绩单都会不由自主的问一句：你的那个科目为什么这么差，但是我觉得我对于那门课还是比较上心的，作业基本是自己完成的，也做了很多课后习题包括笔记，整个脉络我都是很清晰的，我觉得我确实掌握了这门课，因此我觉得这个成绩或许也没有那么重要了(虽然还是很不服气)

    但是说归说，在现在想要获得继续读书的机会还是需要得到一个好的GPA去保研或者考研/申请，这样才有机会去继续我的学业，所以为了我的科研理想也只好去吃石了

    我现在刚好正在等待这门课出分后我排名的震动，我现在的心情几乎不亚于高考出分，但是我觉得，我应当是做了我能做的一切了

    后编：考的最差的一门拉低了我几乎0.1的GPA，然而我的排名只掉了3个人，大喜！

    脱离了迷茫下面再来详细的说一说今年学到的一些数学/统计方面的理论知识：

    Statistic/Math Theory Learning

    今年主要研究的大概就是高维统计理论了，学习了法国的一本很著名的书HDS，不知道为什么现在很难下载到了，可能是版权出问题被投诉下架了

    2024年度总结

    2024年度总结

    前言

    发现很久没有系统地回顾自己的学习生活了，刚好期末快考完了，所以极限来知乎水一篇年度回答(bushi)。鉴于我本人水平比较菜，也就是竭力伪装成一个正常大学牲的普通一年生活(请给出p值doge)，只是一点日常记录看看就好了，不要骂我不努力，万分感谢。

    正文

    背景

    本人某9大二普通学生，从去年(现在或许是前年了？虽然我写的时候还是去年)上大一以来就一直忙着混过一些必修课考试并抓准时机做补番和打游戏诸如此类的事情，没有做大量超出原本数分高代解几之类的学习。

    小总结

    到2024年来学业和生活还算有了些起色，先是顶着语言debuff学习了一些英文教材和课程(但是回过头也不敢肯定自己学的多扎实)，然后是学习(折腾)了一些LaTeX和markdown相关的软件和操作，能用LaTeX写一些简单课程论文和课程笔记或许是我今年第二大的收获？第一大收获大概是对自己的英文水平有了一些自信，敢于去阅读一些上古英文文献和一些英文资料，上英文社区寻找资料了(科技点亮世界，而世界互相帮助的感觉真的很好)。

    因为个人还算兴趣比较广博的人，2024上半年可能什么都摸了摸(拓扑，实变，概率论)然后都感觉挺有意思的，看了些测度论和范畴论相关的(毕竟不需要什么门槛)，但是对于在知乎上甚嚣尘上的代数几何和PDE之类神秘古神语也敬而远之(前置知识也不是很够不太敢去碰，也不想当名词党，也不是那种擅长用什么学什么的人)，再然后就是发现对于概率论比较有兴趣，并且了解一番发现概率论需要建立在测度之上后有了一些冥冥之中的感觉，决定下一年要好好学习一些相关的理论。

    说到好好学习这里可能比较值得宕开一笔好好说道说道

    吐槽部分

    Warning!!! 以下内容有很强的个人情绪

    TMD，上个大学这么多事！好多无聊无意义的事情啊，我确实不是什么热衷社交的人，平常的爱好也只是看看番，打打游戏，顶多加上看数学书。我这种zz鼠鼠平常出门都恨不得走下水道，这么多推不掉的事属实震撼到我了，也是小小的变e了，常常思考别人的大学是否也是这么多事，一问都一个样，噫，乐！

    回归正文

    对于好好学习这件事还是有一些看法，刚上大学的人(me yes)可能还有比较多学生思维吧(随便怎么叫吧)，具体大概表现就是不怎么敢翘课，然后想着每门课的成绩怎么保，怎么拿高分之类的，心里完全知道这些事可能并不是很有利于数学学习，然后就是一整个焦虑，而且大环境也很卷，总是有相关的保研和考研言论绑架，搞得不读研读博跟杀人放火了一样。然后看数学书也不是特别有经验，经常看到一半发现自己好像上面没看懂呢就给看下面了，然后看不懂一直打转压力更大了(现在感觉就是对于那套语言不是很习惯，也不是很能把握学习的重点，看书搁那验证证明，思考证明思路思考的太少了，一句话就是看书不带脑子)。

    这学期就感觉好多了，选择性上了很多课，有了很多收获。虽然不是我GPA高了，是我番看多了，游戏时长进步了，但是变得开心了，生活之余也经常有人劝我不焦虑的，私下认为这也是很难做到的，怎么可能不去思考自己的未来呢？但是我摆着摆着还真摆明白了，之前感觉有点云里雾里的东西现在看居然十分的自然，之前感觉肯定做不出的题，会在自己十分有耐心的抽丝剥茧中慢慢解决，这种一切完美闭环的感觉才是我一开始追求的美好，而不是什么“你GPA多少？”。数学书是有点明白怎么看的了，问题是有一些想法的了，感觉真的很棒。

    既然有了些经验就要利用一下了，我觉得我也终于可以开始做一些更深的数学探索了，那些快乐且充满希望的时光仿佛又回来了，希望明年能少摆点烂。

    碎碎念

    其他的调剂大概就是在业余生活之中加了一些数学群，但是发现现在的数学群氛围大都不是特别好，动不动就是mol来mol去的，不是很感冒，也没什么意思，毕竟显然我也不是什么能被mol的玩意，遂摆之。

    然后就是跟数学不太相关的部分了，算是加了一些学弟?(确实没有学妹)，发现直系高中的学弟比我见到的大多数大学生都正常(由于是死宅无法根据身边统计学得出可信度较高的结论)，帮助答疑了一些问题，再然后就是把之前打算在大一接触的一些东西翻了出来打算慢慢开始学(主要是csdiy上的一些东西)，虽然不打算转码但是确实对这些看起来很有趣的东西没有什么抵抗力，也建了一个自己的个人网站玩算是一个纪念吧。
